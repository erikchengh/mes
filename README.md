# 制药工艺流程MES系统

## 概述
这是一个专为制药行业MES（制造执行系统）专业人员设计的工艺流程对比和分析系统。系统涵盖了完整的制药工艺门类，提供了专业的GMP合规性检查、风险评估、批记录分析等功能。

## 功能特点

### 1. 完整制药品类覆盖
- 原料药生产工艺
- 制剂生产工艺（固体制剂、注射剂、半固体制剂、液体制剂）
- 生物制品生产工艺（疫苗、血液制品）
- 新型制剂生产工艺（靶向制剂、透皮给药系统等）

### 2. MES专业功能
- 工艺风险评估（基于ICH Q9）
- GMP合规性检查
- 批记录管理与分析
- 设备综合效率(OEE)计算
- 参数趋势分析

### 3. 专业可视化
- 工艺流程图（GMP标准）
- 风险评估图表
- 多产品对比雷达图
- 参数趋势监控图

### 4. 用户体验优化
- 专业暗色主题（适合长时间工作）
- 响应式设计
- 模块化架构
- 实时数据模拟
